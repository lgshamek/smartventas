Comprobantes y SUNAT

Comprobantes y gestion para SUNAT

SmartVentas cuenta con facturacion electronica para negocios del Regimen MYPE Tributario y del Regimen Especial, segun la configuracion disponible en el sistema. Para negocios del Nuevo RUS, permite registrar la venta y descargar los datos necesarios para emitir manualmente la boleta desde SUNAT Operaciones en Linea.

Venta registrada

El sistema guarda cliente, productos, pagos, fecha y comprobante.

>

Comprobante

Boleta, factura o ticket con datos listos para imprimir o procesar.

>

SUNAT

Envio configurado o exportacion de datos para gestion manual cuando corresponda.
